# Computer Systems: A Programmers Perspective (CS:APP)
The goal for thie repository is to work through [Computer Systems: A Programmers Perspective 3rd Edition](https://csapp.cs.cmu.edu) and hopefully get better at understanding computers and how they work from a software perspective. I often write software without really knowing how the high level concepts I work with map to the lower level ones that actually drive the system so I hope to change that.
  
This repo is a collection of my notes, coursework, and practice problems as I work through this text.

I have found that many of the practice problem solutions in the textbook do not show the work and steps taken to get the solution so I will try to focus on making my solutions readable and followable for others and myself in the future.
