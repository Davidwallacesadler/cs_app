# Practice 2.30

Write a function with the following prototype:

```c
/* Determine whether arguments can be added without overflow */
int tadd_ok(int x, int y);
```

The function should return 1 if arguments x and y can be added without causing overflow.

---

So we want to have a function that checks if adding two two's-complement integers together will overflow.

We know that there can be the following base cases for two's completement sums from the chapter (given x, y, and s for the sum):
1. x > 0, y > 0, s > 0 => Normal Positive Operation
2. x > 0, y > 0, s < 0 => Positive Overflow
3. x < 0, y < 0, s < 0 => Normal Negative Operation
4. x < 0, y < 0, s >= 0 => Negative Overflow

So we want our function to check for the cases 2. and 3. 

```c
int tadd_ok(int x, int y) {
    int sum = x + y;

    // Check if we overflowed either way
    if ((x > 0 && y > 0 && sum < 0) || (x < 0 && y < 0 && sum >= 0))
        return 0;
    
    return 1;
}
```
