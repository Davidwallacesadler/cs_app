| [A.](#a)| -1      | ___     | 
| [B.](#b)| -5      | ___     |

## A.

T2U(-1):

```
= -1 + 2^4
= -1 + 16
= 15
= 0x1111 
= 0xF
```

## B.

T2U(-5):

```
= -5 + 2^4
= -5 + 16
= 11
= 0x1011
= 0xB
```

## C.

T2U(-6):

```
= -6 + 2^4
= -6 + 16
= 10
= 0x1010
= 0xA
```

## D.

T2U(-4):

```
= -4 + 2^4
= -4 + 16

= 0x1100
= 0xC
```

