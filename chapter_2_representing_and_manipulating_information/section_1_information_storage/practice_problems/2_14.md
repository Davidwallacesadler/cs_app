# Practice 2.14

Suppose `a` and `b` have byte values `0x55` and `0x46`, respectively. Fill in the following table indicating the byte values of different C expressions:

| row     | Expression | Value | Expression | Value |
| ------- | ---------- | ----- | ---------- | ----- |
| [A](#a) | a & b      | ___   | a && b     | ___   |
| [B](#b) | a | b      | ___   | a || b     | ___   |
| [C](#c) | ~a | ~b    | ___   | !a || !b   | ___   |
| [D](#d) | a & !b     | ___   | a && ~b    | ___   |

---

## A.

`a & b`:

```
0x55 = 0101 0101
0x46 = 0100 0110

0101 0101
0100 0110 &
-----------
0100 0100 = 0x44
```

`a && b`:

```
0x55 && 0x46 
=> is 0x55 zero? No.
=> 0x01
```

## B.

`a | b`:

```
0101 0101
0100 0110 |
-----------
0101 0111 = 0x57
```

`a || b`:

```
0x55 || 0x46 = 0x01
=> is 0x55 zero? No.
=> 0x01
```

## C.

`~a | ~b`:

```
0x55 = 0101 0101
0x46 = 0100 0110

~0x55 = 1010 1010 = 0xAA
~0x46 = 1011 1001 = 0xB9

1010 1010
1011 1001 |
-----------
1011 1011 = 0xBB
```

`!a || !b`:

```
!0x55 || !0x46 
=> 0x00 || 0x00
=> 0x00
```

## D.

`a & !b`:

```
0x55 = 0101 0101
!0x46 = 0x00

1010 1010
0000 0000 &
-----------
0000 0000 = 0x00
```

`a && ~b`:

```
0x55 && ~0x46
=> 0x55 && 0xB9
=> is 0x55 zero? No.
=> 0x01
```
